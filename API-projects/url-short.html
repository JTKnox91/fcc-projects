URL Shortener

Creating a new Short URL

Example Usage:

https://jtknox91-fcc.herokuapp.com/url-short/n/http://www.google.com

JSON Output:

{ "original_url": "http://www.google.com", "short_url": "https://jtknox91-fcc.herokuapp.com/url-short/s/AbCd" }

Using a Short URL

Example Usage:

https://jtknox91-fcc.herokuapp.com/url-short/s/AbCd

Now redirects to:

http://www.google.com
